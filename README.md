# Тест-кейсы

## Тест-кейс 1: Создание клиента (Add Customer)

**Цель:** Проверить корректность добавления клиента с автоматически сгенерированными данными.

**Данные:**
- Post Code: динамические 10 цифр (пример: 0001252667)
- First Name: вычисляется из Post Code:
    - Каждую цифру преобразовываем в букву английского алфавита по порядку от 0 до 25.
    - Если цифра больше 25, то начинаем с 26 как с 0. Т.е. 0 - a, 26 - тоже a, 52 – тоже a, и т.д.
- Last Name == First Name

**Шаги:**
1. Открыть вкладку Add Customer
2. Ввести:
    - First Name
    - Last Name
    - Post Code
3. Нажать Add Customer
4. Подтвердить появившийся alert: Customer added successfully with customer id : <номер>
5. Перейти во вкладку Customers
6. Ввести Post Code в поле поиска
7. Проверить, что клиент отображается в таблице

**Ожидаемый результат:**
- Отображается alert: Customer added successfully with customer id : <номер>
- Клиент с введенными данными присутствует в таблице


## Тест-кейс 2: Сортировка клиентов по имени (First Name)

**Цель:** Проверить корректность сортировки клиентов по столбцу First Name

**Предусловие:** В таблице Customers >= 3 записей

**Шаги:**
1. Открыть вкладку Customers
2. Сохранить текущий порядок имён
3. Нажать заголовок First Name
4. Проверить порядок имен от a до Z
5. Нажать заголовок повторно
6. Проверить порядок имен из п. 2

**Ожидаемый результат:**
- После первого клика сортировка по возрастанию от a до Z
- После второго вернется в исходное состояние
- Порядок и строк изменяется корректно, а колличество не изменяется


## Тест-кейс 3: Удаление клиента

**Цель:** Проверить удаление клиента, длина имени которого ближе всего к среднему арифметическому длины всех имён

**Предусловие:** В таблице Customers >= 3 клиента

**Шаги:**
1. Сохранить все значения First Name
2. Узнать длину каждого имени
3. Найти среднее арифметическое длин имен
4. Удалить клиента с именем ближе всего к среднему арифметическому
5. Проверить обновлённый список клиентов:
    - Имя удалённого клиента отсутствует
    - Все остальные имена из п. 1 присутствуют в списке

**Ожидаемый результат:**
- Удалён именно клиент длина имени которого ближе всего к среднему арифметическому длины всех имён
- Количество строк уменьшилось на 1
- Остальные клиенты остались без изменений
